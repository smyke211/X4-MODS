Getting Started: Tools, Scripting and Modding

Author: euclid (Moderator) | Date: Fri, 30. Nov 18

Unpacking Game Files

Use XRCatTool from X Rebirth (works for X4).

Download and place in X4 root directory (where X4.exe is).

Example unpack.bat

E:
cd "Games\Steam\SteamApps\common\X4 Foundations"
XRCatTool.exe -in 01.cat -in 02.cat -in 03.cat -in 04.cat -in 05.cat -in 06.cat -in 07.cat -in 08.cat -in 09.cat -out "E:\Games\Steam\SteamApps\common\X4 Foundations\unpacked"
pause

Useful Tools

  • AstroGrep — find entries in game files
  • Notepad++ or Visual Studio Express — XML editing

Mission Director (MD) and XML Script Commands

After unpacking, find .xsd files in folders:

  • libraries/
  • md/
  • scripts/

Also check scriptproperties.html in unpacked folder root — contains all expressions defined in XSD files.

Activate Mods

  1. Create extensions/ folder in X4 root
  2. If root doesn't work, try X4 document folder instead
  3. Create mod folder inside extensions
  4. Create content.xml with metadata
  5. Create md/ or aiscripts/ folders for scripts

Example content.xml

<?xml version="1.0" encoding="utf-8"?>
<content id="PJ" name="PlayerJump" 
  description="Player Ship can Jump to a Target via Autopilot"
  author="Euclid" version="0.1" date="2018-15-09" save="0" enabled="1">
  <text language="44" name="PlayerJump" 
    description="Player ship can jump via autopilot to a selected target"
    author="Euclid" />
</content>

Tips

  • Backup your saved games before testing mods
  • Use debuglog to check extension loading
  • Check in-game "extensions" settings to confirm mod is active
