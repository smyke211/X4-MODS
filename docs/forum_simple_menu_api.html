[MOD/API] Simple Menu API

Author: SirNukes | Date: Thu, 14. Nov 19

Overview

Adds support for generating menus using mission director scripts in X4.

A lua backend handles the details of interfacing with the x4 ui system.

Two Types of Menus

  • Options menus — integrated with standard x4 options
  • Standalone menus — display immediately

Requirements

  • Lua Loader API extension

Usage Patterns

1. Basic Extension Options

Call Register_Option with:

  • name
  • default value
  • widget type
  • callback (receives new value on change)

2. Standalone Menu

  • Call Create_Menu to open new menu (closes others)
  • Fill with rows and widgets
  • Specify callback cues
  • Callbacks signalled on player input

3. Options Menu

  • Call Register_Options_Menu to register with backend
  • Create cue to make rows and widgets
  • Set cue as onOpen callback target
  • Access through "Extension Options" page

Supported Widgets

From community note (teleportationwars):

  • Icons, scaling, resizing, colors
  • Edit boxes, dropdowns, buttons
  • Partial redrawing, scrolling, fonts
  • Sliders, checkboxes

Example — Simple Option

<cue name="Reset_On_Reload" instantiate="true">
  <conditions>
    <event_cue_signalled cue="md.Simple_Menu_Options.Reloaded"/>
  </conditions>
  <actions>
    <signal_cue_instantly cue="md.Simple_Menu_Options.Register_Option" 
      param="table[
        $id = 'debug_menu_api',
        $name = 'Enable menu api debug logs',
        $mouseover = 'Prints extra api status info to the debug log',
        $default = 0,
        $type = 'button',
        $callback = OnChange,
      ]"/>
  </actions>
</cue>

Example — Standalone Menu

<cue name="Open_Menu" instantiate="true" namespace="this">
  <conditions>
    <event_cue_signalled/>
  </conditions>
  <actions>
    <signal_cue_instantly cue="md.Simple_Menu_API.Create_Menu" 
      param="table[$columns = 1, $title = 'My Menu', $width = 500,]"/>
    <signal_cue_instantly cue="md.Simple_Menu_API.Add_Row"/>
    <signal_cue_instantly cue="md.Simple_Menu_API.Make_Label" 
      param="table[$col=1, $text='Hello world']"/>
  </actions>
</cue>
