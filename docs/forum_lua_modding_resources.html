Lost on Lua Modding Resources

Author: Aniketos | Date: Thu, 14. Mar 24

The Problem

Lua modding documentation is sparse and scattered. Old Confluence wiki had more info but links no longer work.

Unanswered Questions

Basic functions lack documentation:

  • DebugError() — what are its parameters?
  • RegisterEvent() — what is its signature?
  • AddUITriggeredEvent() — can it be handled with RegisterEvent?

Where to Look

  • X Rebirth Wiki has UI modding support page with Lua functions subpage
  • Widgetsystem function overview (for RegisterEvent)
  • Core Lua scripts in game files
  • Published mods' code

The Reality

Lua modding is difficult to break into due to:

  • Lack of introductory material
  • Incomplete documentation
  • No central reference for core library functions

Advice

Perseverance through trial and error, reading core scripts and mod code.

If you reach competence, consider contributing documentation.
